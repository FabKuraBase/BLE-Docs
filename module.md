# BLEモジュール

BLEモジュールは、国内では「特定無線設備の種別	第２条第１９号に規定する特定無線設備」として認証される。電波の型式「Ｆ１Ｄ」、周波数は、「2402～2480ＭＨz(２ＭＨz間隔40波)」と定義される。
